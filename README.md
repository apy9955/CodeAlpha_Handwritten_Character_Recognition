# CodeAlpha Handwritten Character Recognition

## Objective
Identify handwritten digits using Deep Learning and Convolutional Neural Networks (CNN).

## Dataset
MNIST Dataset (60,000 training images and 10,000 testing images)

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

## Model
Convolutional Neural Network (CNN)

## Results
- Test Accuracy: 99.03%

## Files
- Handwritten_Character_Recognition_Project.ipynb
- handwritten_character_model.h5
- Handwritten_Character_Report.csv
